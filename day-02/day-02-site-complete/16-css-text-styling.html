Text-Transform

text-transform is used to set the capitalization of the text. You can set text transform to the following values:

  • none: default, no capitalization applied
  • capitalize: sets the first letter of each word to uppercase
  • uppercase: sets the letters to all uppercase
  • lowercase: sets the letters to all lowercase

Text-Decoration

text-decoration is used to set special styling on text. You can set text-decoration to the following values:

  • none: no text decoration applied. It is often used to overide default text-decoration of links
  • overline: creates a line above the text
  • underline: creates a line below the text. In general you want to only use underlines in web design if the text is a link
  • line-through: create a line that runs through the middle of the text

Text-Align

text-align is used to set the alignment of text.

  • left: aligns text to the left (default)
  • right: aligns text to the right
  • center: aligns text to the center of its parent container
  • justify: stretches each line of text so that each line of text has the same width

Word-Wrap

word-wrap allows you to set how long words will wrap. The default is for long words not to wrap accept at allowed break points (normal).

  • normal: break words only at allowed break points
  • break-word: allows long words to break on to multiple lines

Text-Overflow

text-overflow property is used to determine how to show to the user that there is more text available. The default is just tp clip the text.

  • clip: Just clips the text. This is the default value
  • ellipsis: Adds a "..." show the user that more text is present

By default the text is clipped, change this so that the overflowed text is represented by an ellipsis. You will have to set the overflow property on the paragraph tag to hidden and set the white-space property to nowrap.
